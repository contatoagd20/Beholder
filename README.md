# Getting Started

## Prerequisites

* Django 2.0.7
* Python 3
* Env
* Pillow

## Installing

Uses Env
```
python3 -m venv myvenv
```

How to access env
```
source myvenv/bin/activate
```

Django

```
pip install Django=2.0.7
```

Pillow
```
pip install Pillow
```

## miscellaneous

Settings configured to *America/Sao_Paulo*, if you need change the setting.py file **TIME_ZONE**
